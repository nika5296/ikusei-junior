## Excel / スプレッドシートの構成（読み取り結果）

### シート `出欠管理票_2026`（4月の名簿）

| 列の意味（概略） | 内容 |
|------------------|------|
| A | **氏名**（`名前／日付` 行の下） |
| B | **学年**（例: 中3, 4, 5, 6 など） |
| C 以降（日付の前まで） | **レギュラー日程**（`月` `火` `木` `金` がセル分けで複数入る） |
| 日付行以降の列 | 各レッスン日の出欠（現状 Excel サンプルでは空。本番は ○× 等） |

- 1行目: 月タイトル（例: `4月`）と、各列の **曜日**（木・金・月…の繰り返し）
- 2行目: 見出し（`名前／日付`｜`学年`｜…｜`レギュラー日程`｜…｜**各日付**）

**週回数の目安**: レギュラー日程に **入っている曜日の数**  
（例: `月` のみ → 週1／`月`と`金` → 週2／`月`・`火`・`金` → 週3）

### シート `2026`（年間・別レイアウト）

- 1月分など、**名前・レギュラー日程・日付列・欠席「欠」** などが入る想定のシートです（Excel では 1月の履歴サンプルあり）。

## V1システム（本プロジェクト）との対応

| 出欠表（Excel） | 生徒マスタ（V1） |
|-----------------|------------------|
| A列 氏名 | **氏名**（フォーム設問タイトルと**完全一致**させる） |
| B列 学年 | **メモ** に書くか、必要なら列を増やして保持 |
| レギュラー日程の複数セル | **レギュラー曜日** に `月,木,金` のように **カンマ区切り1セル**へまとめる |
| 曜日の個数 | **コース** を `週1` / `週2` / `週3` に対応づけ |

- **出欠の日付マトリクス**は V1 では **フォーム＋出欠ログ** に置き換えます（表の「意味」は維持し、入力経路だけ変更）。

## マスタ用 CSV の作り方（自動）

PC に Python 3 がある場合、Excel から **生徒マスタに貼り付けやすい TSV** を出せます。

```powershell
py -3 tools\excel_to_master_tsv.py "%USERPROFILE%\Downloads\育成クラス出欠表_2026.xlsx"
```

出力は `tools\master_from_出欠管理票_2026.tsv` に保存されます。  
内容を確認のうえ、**生徒マスタ** にコピペし、`studentId` と `viewToken` を調整してください。

## 運用上の注意

- Google スプレッドシートを **編集用の正** にする場合、Excel は **バックアップ・共有用** として同期運用するとよいです。
- 氏名に **全角スペース**（例: `鈴村　賢信`）が含まれる場合、フォームの見出しと **一字一句合わせる**必要があります。
